# Face Recognition Attendance System

An AI-based attendance management system developed using Python, OpenCV, and Django.

## Project Overview
This project automatically detects and recognizes student faces using a webcam and marks attendance digitally. The system reduces manual attendance errors and improves efficiency.

## Features
- Face detection and recognition
- Automated attendance marking
- Student registration and dataset creation
- Attendance record management
- Admin dashboard support

## Technologies Used
- Python
- OpenCV
- Django
- NumPy
- Pandas
- SQLite

## Folder Structure
- facerec/ → Face recognition modules
- dataset/ → Student image datasets
- templates/ → HTML templates
- static/ → CSS, JS, and assets
- db.sqlite3 → Database file
